    Welcome to the Page for a 3D Game Thing.

    This project is a small 3D Game Environment that I use to test out developing 3D games.

    Typically this is used to test out concepts.

    It is still in development and as such it is still continuously updated. The build and web demo are updated at the end of each day where work was done on it.

    All assets were created by Unity Technologies.
